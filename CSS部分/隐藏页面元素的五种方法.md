# 隐藏页面元素的五种方法

## 设置宽高

```css
width: 0;
height: 0;
```

## Opacity

设置透明度为0

## visibility

设置可见属性为hidden

## position: absolute

```css
{
  position: absolute;
  top: -9999px;
  left: -9999px;
}
```

## display: none

## clip-path

裁剪元素

```css
.hide {
  clip-path: polygon(0px 0px,0px 0px,0px 0px,0px 0px);
} 
```
